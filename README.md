# 📿 Tesbih

**A digital tasbih (prayer bead counter) with dhikr tracking** — tap to count, fully offline PWA.

## Quick Start

Open `index.html` in any browser — works offline after first load.

## Files

| File | Description |
|------|-------------|
| `index.html` | Main app |
| `tasbih.html` | Alternate/legacy version |

## Features

- Tap or click to count
- Dhikr presets (SubhanAllah, Alhamdulillah, Allahu Akbar)
- Progress tracking with daily goals
- Installable PWA — works fully offline
- Clean, minimal interface
